# 🫙 ארבע הקופות

אפליקציית כסף לילדים: ארנק אחד שאליו נכנס כסף, וממנו מחלקים לארבע צנצנות —
**פינוקים**, **חיסכון קצר** (5% ריבית חודשית), **השקעה לטווח ארוך** (10% ריבית חודשית), ו**נתינה** (אמא מכפילה כל שקל).
ההשקעה הארוכה כוללת גם מס של 25% על הרווח במשיכה לפני שנה — בדיוק כמו בחיים.

אפשר גם להרוויח כסף דרך **מטלות בית** — ההורה מגדיר מטלות עם תגמול, והכסף שנצבר נכנס לארנק לחלוקה לצנצנות.

האפליקציה עובדת גם בלי אינטרנט, נשמרת על המכשיר, וניתנת להתקנה על מסך הבית (PWA).

---

## הקבצים

| קובץ | תפקיד |
|---|---|
| `index.html` | האפליקציה עצמה |
| `manifest.webmanifest` | הגדרות ההתקנה (שם, אייקון, צבעים) |
| `sw.js` | עבודה ללא אינטרנט |
| `icon-192.png`, `icon-512.png` | אייקונים (אנדרויד) |
| `apple-touch-icon.png` | אייקון לאייפון בלבד — לא חובה |

> חשוב: כל הקבצים צריכים לשבת **באותה תיקייה**, בשורש (בלי תיקייה עוטפת).

---

## שלב 1 — להעלות לאינטרנט (GitHub Pages)

1. ב‑[github.com](https://github.com) → **New** ליצירת repository חדש. שם לדוגמה: `kupot`, סמנו **Public**, ולחצו **Create repository**.
2. בעמוד הריק לחצו על **uploading an existing file**.
3. גררו את כל הקבצים לתוך החלון ולחצו **Commit changes**.
4. **Settings** → בתפריט הצד **Pages**.
5. תחת *Branch* בחרו **main** ו‑**/(root)** → **Save**.
6. המתינו דקה ורעננו — יופיע קישור בסגנון:
   `https://<שם-המשתמש>.github.io/kupot/`

### חלופה מהירה בלי GitHub
גררו את התיקייה ל‑[app.netlify.com/drop](https://app.netlify.com/drop) ותקבלו קישור מיידי.

## שלב 2 — להתקין על האנדרויד

1. פתחו את הקישור ב‑**Chrome** בנייד.
2. תפריט שלוש הנקודות **⋮** למעלה מימין.
3. בחרו **"התקנת אפליקציה"** (או "הוספה למסך הבית").
4. אשרו **התקנה**.

האייקון יופיע על מסך הבית והאפליקציה תיפתח במסך מלא.
אם לא מוצע "התקנה" — רעננו פעם אחת ונסו שוב.

## איך משתמשים

1. **➕ קיבלתי כסף** — רושמים סכום שנכנס לארנק.
2. **✂️ לחלק לצנצנות** — מחלקים את כסף הארנק לצנצנות (אפשר גם "⚖️ שווה").
3. נכנסים לכל צנצנת כדי לראות פירוט ו‑**➖ להוציא** (עם הסבר).
4. **🎉 ריבית** פעם בחודש — מוסיף 5% לחיסכון ו‑10% להשקעה על היתרה הנוכחית.

צנצנת מתמלאת **רק** מחלוקה של כסף שנרשם — אין הוספה חופשית.

## מטלות בית 🧺

דרך תפריט **☰** (למעלה מימין) עוברים בין **הקופות** ל**מטלות בית**.

1. ההורה מוסיף מטלות עם **➕ מטלה חדשה** — שם, אייקון, כמה מקבלים, ו**כל יום** או **כל שבוע**.
2. הילד/ה מסמן/ת **עשיתי! 🙌** על מטלה שביצע/ה.
3. ההורה מאשר/ת (**✓ אישור**), והתגמול נכנס **לארנק** — ומשם מחלקים לצנצנות כרגיל.

המסך מציג **יום‑יום** עם התאריך האמיתי. אפשר לדפדף אחורה בעזרת החיצים **‹ ›** כדי לראות מה בוצע בימים קודמים (לצפייה בלבד — מסמנים ומאשרים רק על היום). מטלות שבועיות מופיעות בקטע נפרד — **"משימות השבוע"**.
